# Big Clock

A simple, large digital clock designed for time management during presentations, classes, or personal work sessions.

## Features

- **Flexible Time Formats**: Toggle between 12-hour and 24-hour display modes.
- **Adjustable Visibility**: Choose from multiple font sizes (Small, Normal, Large, XL) to suit your screen size and viewing distance.
- **Minimalist Design**: A clean, distraction-free interface that focuses solely on the time.
- **Visual Customization**: Includes a custom clock favicon for easy tab recognition.
